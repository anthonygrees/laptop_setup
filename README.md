# laptop_setup
A script to automatically set up your new Mac Laptop.  
  
### What does it install ?
The script installs the following:
- homebrew to manage packages
- iterm2
- git
- bash-completion
- jq
- python
- packer
- packer-completion
- terraform
- tree
- chef-workstation
- habitat
- vagrant
- vagrant-completion
- visual-studio-code
- atom
- awscli
- azure-cli
- google-cloud-sdk
- virtualbox
- virtualbox-extension-pack
- firefox
- spotify
- whatsapp
  
### How to run it.
Download the `laptop_setup.sh` to your Mac.  The run the following command:  
```bash
./laptop_setup.sh
```
  
You will need to type in your `Admin` password, then you will see the following:
```bash
x
```
